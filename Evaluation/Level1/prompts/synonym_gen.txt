Generate diverse English expressions for the following biosignal parameter as it would naturally be used by three different audiences: clinical doctors, data scientists, and laypersons.

PARAMETER:
  param_key      : {param_key}
  semantic_name  : {semantic_name}
  unit           : {unit}
  category       : {concept_category}

OUTPUT RULES:
- "direct"       : The raw param_key itself and any common shorthand variants (e.g., device prefix stripped).
- "semantic_en"  : Full English clinical/physiological expressions a doctor or nurse might say.
- "medical_term" : Precise medical or pharmacological terminology, including condition names that imply this parameter.
- "abbreviation" : Short abbreviations used on monitors, in papers, or in clinical notes.
- Each list must have 2–6 unique, non-overlapping entries.
- Do NOT repeat the same string across categories.
- Do NOT include Korean or any non-English expressions.

OUTPUT FORMAT (valid JSON only, no markdown):
{{
  "direct":       ["...", "..."],
  "semantic_en":  ["...", "..."],
  "medical_term": ["...", "..."],
  "abbreviation": ["...", "..."]
}}
